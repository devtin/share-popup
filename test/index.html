SharePopUp - v1.1

A simple JavaScript wrapper for sharing content on different social networks, including: Twitter, Facebook, Pinterest, LinkedIn, Tumblr, Reddit and Google+. Now also available via Whatsapp and SMS.

Usage

  • Twitter
  • Facebook
  • Pinterest
  • LinkedIn
  • Google+
  • Tumblr
  • Reddit
  • E-mail
  • Whatsapp
  • SMS

Demo Variables

var url     = "http://devtin.io",
    title   = "Professional Lego Player",
    text    = "Have a look at this website: "+url,
    image   = "http://devtin.io/wp-content/uploads/2015/03/logo-web-01-150x150.png",
    email   = "tin@example.com",
    cc      = "cc@example.com",
    bcc     = "bcc@example.com";

Twitter

SharePopUp.twitter ( text [, width , height , blank ] )

Basic Example:

SharePopUp.twitter(text);

Bigger window:

SharePopUp.twitter(text,1024,768);

New window:

SharePopUp.twitter(text,null,null,true);

Facebook

SharePopUp.facebook ( url [, width , height , blank ] )

Email

SharePopUp.email ( to , cc , bcc , subject , body [, width , height , blank ] )

Basic Example:

SharePopUp.email(email,null,null,title,text);

Basic Example with CC and BCC:

SharePopUp.email(email,cc,bcc,title,text);

Bigger Window:

SharePopUp.email(email,cc,null,title,text,1024,768);

New Window:

SharePopUp.email(email,cc,null,title,text,null,null,true);

Whatsapp

SharePopUp.whatsapp ( text )

Please note that this function won't perform any validation regarding the device where the app is running or if Whatsapp is installed or not.

Basic Example:

SharePopUp.whatsapp(text);

SMS

SharePopUp.sms ( text [, phoneNumber ] )

Please note that this function won't perform any validation regarding the device where the app is running or wether if the device can send SMS's or not.

Basic Example:

SharePopUp.sms(text);

Basic Example:

SharePopUp.sms(text,'+1.305.123.4567');
